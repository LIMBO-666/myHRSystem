# 人力资源管理系统 - 运行指南

## 🚀 前端运行（无需后端）

### 方法一：使用Node.js（推荐）

1. **打开IDEA Terminal**
   - 快捷键：`Alt + F12`
   - 或点击底部 `Terminal` 标签

2. **进入ui目录并启动服务器**
   ```bash
   cd ui
   npx http-server -p 8000 -o
   ```

3. **访问系统**
   - 浏览器会自动打开
   - 或手动访问：`http://127.0.0.1:8000/login.html`

4. **停止服务器**
   - 在Terminal中按 `Ctrl + C`

### 方法二：直接打开文件

1. 在文件管理器中找到 `ui` 文件夹
2. 双击 `login.html` 文件
3. 浏览器会自动打开

### 方法三：使用Python

```bash
cd ui
python -m http.server 8000
```

然后访问：`http://localhost:8000/login.html`

## 🔧 后端运行（可选，当前未实现）

### 前提条件
1. 安装JDK 17或更高版本
2. 安装Maven
3. 安装MySQL并导入数据库

### 运行步骤

1. **导入数据库**
   - 参考 `数据库导入说明.md`
   - 或使用IDEA Database工具执行 `storage/src/main/resources/database/myHRSystem.sql`

2. **配置数据库连接**
   - 修改 `authorization-management/src/main/resources/application.yml`
   - 设置正确的数据库密码

3. **启动后端服务**
   ```bash
   # 在项目根目录
   mvn clean install
   cd authorization-management
   mvn spring-boot:run
   ```

4. **关闭Mock API**
   - 修改 `ui/js/mock-api.js`
   - 将 `USE_MOCK = false`

## 📝 注意事项

1. **Mock模式**：默认启用，无需后端即可运行
2. **数据持久化**：数据保存在浏览器LocalStorage中
3. **切换真实API**：修改 `ui/js/mock-api.js` 中 `USE_MOCK = false`
4. **重置数据**：访问 `ui/重置数据.html` 或清除浏览器缓存
