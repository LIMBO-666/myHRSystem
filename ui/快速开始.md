# 🚀 快速查看前端效果

## 方法一：直接打开（最简单）

1. **打开演示页面**
   - 双击 `demo.html` 文件
   - 或右键选择"打开方式" → 选择浏览器

2. **直接登录**
   - 双击 `login.html` 文件
   - 使用测试账号登录：
     - 用户名：`hr01`
     - 密码：`password123`

## 方法二：使用本地服务器（推荐）

### 使用Python（如果已安装）
```bash
# 进入ui目录
cd ui

# 启动服务器
python -m http.server 8000
```

然后访问：`http://localhost:8000/demo.html`

### 使用Node.js（如果已安装）
```bash
# 进入ui目录
cd ui

# 启动服务器
npx http-server -p 8000
```

然后访问：`http://localhost:8000/demo.html`

### 使用VS Code Live Server扩展
1. 安装 "Live Server" 扩展
2. 右键点击 `demo.html` 或 `login.html`
3. 选择 "Open with Live Server"
