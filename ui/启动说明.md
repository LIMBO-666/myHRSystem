# 在IDEA中启动前端服务器

## 方法一：使用Node.js（推荐）

### 步骤：
1. 在IDEA中打开项目
2. 打开底部 Terminal（终端）窗口
   - 快捷键：`Alt + F12`（Windows/Linux）
   - 或点击底部工具栏的 `Terminal` 标签
3. 进入ui目录：
   ```bash
   cd ui
   ```
4. 启动服务器：
   ```bash
   npx http-server -p 8000 -o
   ```
   - `-p 8000`：指定端口8000
   - `-o`：自动打开浏览器

### 停止服务器：
在Terminal中按 `Ctrl + C`

---

## 方法二：使用Python

### 步骤：
1. 打开IDEA的Terminal
2. 进入ui目录：
   ```bash
   cd ui
   ```
3. 启动服务器：
   ```bash
   python -m http.server 8000
   ```
   或
   ```bash
   python3 -m http.server 8000
   ```
4. 手动打开浏览器访问：`http://localhost:8000/login.html`

---

## 方法三：使用IDEA内置服务器（最简单）

### 步骤：
1. 右键点击 `ui/login.html` 文件
2. 选择 `Open in Browser` → 选择浏览器
3. 或者直接双击 `login.html` 文件

**注意**：这种方式可能会有跨域问题，如果遇到问题，请使用方法一或方法二。

---

## 推荐配置

建议在IDEA中配置一个运行配置：

1. 点击右上角的运行配置下拉菜单
2. 选择 `Edit Configurations...`
3. 点击 `+` → 选择 `Node.js`
4. 配置：
   - **Name**: `启动前端服务器`
   - **Node interpreter**: 选择你的Node.js路径
   - **Node parameters**: 留空
   - **Working directory**: `$PROJECT_DIR$/ui`
   - **JavaScript file**: `node_modules/.bin/http-server`（需要先安装）
   - **Application parameters**: `-p 8000 -o`

或者更简单的方式，直接在Terminal中运行命令。
